# Mambixar v1.0 🌸

Welcome to Mambixar, a lightweight, ultra-high-speed, zero-dependency scripting language designed for elegant, linear logic flow. I hope this becomes a mainstream language.

## Features
* **Zero Overhead:** Preallocated memory structures for blazing-fast execution.
* **Deterministic Sorting:** Built-in structural optimization handles execution queuing natively.
* **Minimalist Syntax:** No boilerplate, no formatting bloat—just pure code.

## Getting Started

Mambixar features a completely universal, graphical interface launcher that runs flawlessly across Windows, macOS, Linux, and mobile Python environments without any manual command-line configuration.

### Setup
1. Download `run_mambixar.py` and the core environment runtime `zzz.pyc` from our Releases page.
2. Keep both files together in the same folder on your device.

### Running Code
You do not need to use the terminal or navigate paths. 

* **On PC:** Simply double-click `run_mambixar.py`. A visual file window will pop up automatically. Select your `.mbx` file, and it will execute instantly.
* **On Mobile:** Open `mambixar.py` in your local Python app and press the **Run/Play** button. Use the on-screen file selector to choose your `.mbx` file.
